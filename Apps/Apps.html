Canvas

Canvas is a simple image drawing and editing web application made using HTML5. As illustrated in the image made with Canvas above, it offers freedrawing, color selection, resizable brush width, saving and uploading capabilities, among more! In addition, all icon graphics were done by yours truly.

Ebay Searcher

Ebay Searcher is a web application that displays ebay listings under a certain keyword reformatted to show the most applicable information. By selecting different column headers, the rows will update to show the listings in ascending or descending order by that category.
